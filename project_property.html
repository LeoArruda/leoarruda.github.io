The core functionality of this Real Estate Platform to kick start my project. The code for a web app that let's users upload their properties and launch your app in days rather than months.

The platform I built, allows users to browse existing properties, create an account, sign in and post their own properties. Users can visit their own dashboard to see their properties and scheduled viewings.


Functionality included:


- Browse existing properties

- Sign up / sign in / forgot password

- Upload new properties

- User dashboard to view properties/viewings
